One-sentence problem: Can we recover expressive SMPL-X body, hands, and face from a single image in one stage, without separate body/hand/face expert networks and late copy-paste fusion?
一句话问题：能否从单张图像一步恢复 SMPL-X 的身体、双手和面部，而不是先分别跑 body/hand/face 专家网络再做后期拼接融合？
Placement in this bank: OSX belongs under Single-View SMPLX regression -> One-Stage / Whole-Body. It is a feed-forward SMPL-X regressor for a single monocular image, complementary to SMPLify-X, which is optimization-based fitting.
在本库中的定位：OSX 归入 Single-View SMPLX regression -> One-Stage / Whole-Body。它是单张单目图像的前向 SMPL-X 回归器，和优化式拟合的 SMPLify-X 互补。

Key Contributions

  • First one-stage whole-body SMPL-X pipeline: OSX predicts body, hands, face, shape, expression, and camera/global translation in one encoder-decoder model, avoiding separate part networks and manual post-processing.
    首个 one-stage whole-body SMPL-X 管线：OSX 用一个 encoder-decoder 同时预测身体、双手、面部、体型、表情和相机/全局平移，避免独立部位网络和手工后处理。
  • Component Aware Transformer (CAT): A global body encoder uses body tokens to regress coarse body parameters and produce a strong feature map for local components.
    Component Aware Transformer (CAT)：全局 body encoder 使用 body tokens 回归粗粒度身体参数，并为局部部件提供高质量 feature map。
  • High-resolution local decoder: A differentiable feature-level upsample-crop branch extracts higher-resolution hand and face features without running separate hand/face backbones.
    高分辨率局部 decoder：通过可微的 feature-level upsample-crop 提取高分辨率手部和面部特征，而不需要额外 hand/face backbone。
  • Keypoint-guided deformable attention: 2D keypoint references initialize component tokens, then deformable attention samples near each keypoint to improve hands and face while keeping cost manageable.
    关键点引导的 deformable attention：用 2D keypoint reference 初始化 component tokens，再在每个关键点附近做 deformable attention 采样，提高双手和面部精度并控制计算量。
  • UBody dataset: Builds a 1.051M-frame upper-body dataset with 15 real-life scenes, 2D whole-body keypoints, 3D SMPL-X annotations, frame-valid labels, person boxes, and hand boxes.
    UBody 数据集：构建 105.1 万帧 upper-body 数据集，覆盖 15 类真实生活场景，含 2D whole-body keypoints、3D SMPL-X 标注、有效帧标签、人框和手框。

Method / Architecture

single RGB image -> patch tokens + body tokens -> global CAT encoder -> feature upsample + RoIAlign crop -> keypoint-guided local decoder -> SMPL-X mesh

Input / output: one human image in, SMPL-X parameters and a whole-body mesh out. The parameter groups include body joint rotations, body shape, global translation, left/right hand rotations, jaw rotation, and facial expression.

Global encoder: image patches are encoded with learnable body tokens. The body tokens regress global/body parameters while the image tokens retain feature maps for local decoding.

Local component decoder: predicted hand/face boxes crop multi-scale feature maps using differentiable RoIAlign. Component tokens are initialized from predicted 2D keypoints and updated through multi-scale deformable cross-attention.

Losses: end-to-end L1 losses on SMPL-X parameters, 3D whole-body keypoints, projected 2D whole-body keypoints, and left/right hand plus face boxes.

输入 / 输出：输入一张人体图像，输出 SMPL-X 参数和 whole-body mesh。参数包括身体关节旋转、体型、全局平移、左右手旋转、下颌旋转和面部表情。

全局 encoder：图像 patch 与可学习 body tokens 一起编码。body tokens 回归全局/身体参数，image tokens 保留给局部 decoder 使用的 feature maps。

局部 component decoder：根据预测的手/脸框，用可微 RoIAlign 裁剪多尺度 feature maps。component tokens 由预测 2D 关键点初始化，并通过多尺度 deformable cross-attention 更新。

损失：端到端 L1 监督 SMPL-X 参数、3D whole-body keypoints、投影 2D whole-body keypoints，以及左右手和面部 boxes。

Limitations

  • Single-image only: OSX does not use temporal consistency, SLAM, IMU, contact, or world-space constraints, so it can jitter on video and cannot recover metric motion by itself.
    仅单图：OSX 不使用时序一致性、SLAM、IMU、接触或世界坐标约束，因此在视频上可能抖动，也不能独立恢复公制运动。
  • Single-person crop assumption: The paper focuses on a human image/crop rather than all-in-one multi-person detection and mesh recovery.
    单人 crop 假设：论文重点是给定人体图像/crop 的恢复，而不是 all-in-one 多人检测加 mesh recovery。
  • Hands and face remain hard: Even with high-resolution decoding, small, occluded, or motion-blurred hands/faces remain the dominant error source.
    手和脸仍然困难：即使有高分辨率 decoder，小尺度、遮挡或运动模糊的手/脸仍是主要误差来源。
  • Parametric body limit: SMPL-X does not explicitly model clothing, hair, props, object contact, or dense surface deformation.
    参数人体限制：SMPL-X 不显式建模衣物、头发、道具、物体接触或稠密表面形变。

Bottom Line

OSX is the clean one-stage baseline for single-image SMPL-X regression: global body context plus local high-resolution hand/face decoding in one Transformer. For this paper bank, it fills the gap between optimization-based SMPLify-X and later specialized hand/egocentric methods. It is useful as an image-level whole-body regressor, but still needs temporal/world-space extensions for egocentric motion capture.
OSX 是单图 SMPL-X 回归里很清晰的 one-stage baseline：在一个 Transformer 中同时建模全局身体上下文和局部高分辨率手/脸解码。对本论文库来说，它填补了优化式 SMPLify-X 与后续专用手部/自我中心方法之间的空位。它适合作为图像级 whole-body regressor，但若用于 egocentric motion capture，还需要时序和世界坐标扩展。
