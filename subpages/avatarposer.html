AvatarPoser: Articulated Full-Body Pose Tracking from Sparse Motion Sensing

Jiaxi Jiang, Paul Streli, Huajian Qiu, Andreas Fender, Larissa Laich, Patrick Snape, Christian Holz
ETH Zurich · Reality Labs at Meta

arXiv PDF GitHub
One-sentence problem: Can a VR/AR system recover full-body SMPL avatar motion in world coordinates from only three sparse 6-DoF inputs: the headset and two hands/controllers?
一句话问题：VR/AR 系统能否只用头显和双手/控制器这 3 个稀疏 6DoF 输入，恢复世界坐标系下的全身 SMPL 虚拟人动作？
Placement in this bank: precursor to later sparse-HMD body tracking work such as HMD-Poser. It belongs under Ego-to-Exo SMPLX Regression → Sparse Sensor / HMD, even though AvatarPoser itself uses the 22-joint SMPL body skeleton rather than SMPL-X.
在本库中的定位：这是 HMD-Poser 等稀疏 HMD 全身追踪工作的前序代表。归入 Ego-to-Exo SMPLX Regression → Sparse Sensor / HMD，但它本身使用的是 22 关节 SMPL 身体骨架，而不是 SMPL-X。

Key Contributions

  • Three-input full-body avatar tracking: Predicts full-body pose from only the head-mounted display and two hand/controller poses, avoiding pelvis trackers or body-worn IMUs.
    三输入全身虚拟人追踪：只用头显和两个手部/控制器姿态预测全身动作，不依赖骨盆追踪器或身体 IMU。
  • Transformer sequence model: Uses a 3-layer, 8-head Transformer encoder over a 40-frame history of sparse tracker signals, then predicts global orientation and local SMPL joint rotations.
    Transformer 时序模型：对 40 帧稀疏追踪信号使用 3 层、8 头 Transformer 编码器，再预测全局朝向和局部 SMPL 关节旋转。
  • Global motion decoupling via Stabilizer: Learns pelvis/global orientation separately instead of directly trusting head orientation, reducing floating-avatar artifacts when users rotate their head independently.
    Stabilizer 解耦全局运动：单独学习骨盆/全局朝向，而不是直接相信头部朝向，降低用户独立转头时的漂浮虚拟人问题。
  • Learning + model-based IK: FK supervision keeps the skeleton physically structured, while a post-network IK step adjusts shoulder/elbow rotations so the predicted hands better match tracked hand positions.
    学习模型 + IK 优化：FK 监督保证骨架结构合理，网络后置 IK 调整肩肘旋转，使预测手部更贴近真实追踪的手部位置。

Architecture

3 sparse 6-DoF trackers → 54-dim/frame signal → Transformer Encoder → Stabilizer + local pose MLP → SMPL FK → arm IK refinement
Input: for each tracker, position, linear velocity, 6D rotation, and angular velocity. With S=3 trackers, each frame has 54 features. The model consumes the previous 39 frames plus the current frame.

Output: 22-joint SMPL pose. The network predicts local joint rotations plus pelvis/global orientation; pelvis translation is inferred from head position through the body kinematic chain.
输入：每个 tracker 包含位置、线速度、6D 旋转、角速度。S=3 时每帧 54 维；模型使用过去 39 帧 + 当前帧。

输出：22 关节 SMPL 姿态。网络预测局部关节旋转和骨盆/全局朝向；骨盆平移通过头部位置和人体运动链推得。

Limitations

  • No visual grounding: Uses only 6-DoF motion signals, so it cannot reason about scene contacts, object interactions, foot visibility, or camera images.
    没有视觉 grounding：只使用 6DoF 运动信号，无法利用场景接触、物体交互、脚部可见性或相机图像。
  • Ambiguous lower body: Inferring legs from head and hands is intrinsically underdetermined, especially for seated poses, small leg motions, or actions where the upper body is weakly correlated with the feet.
    下肢歧义：从头和手推腿部本身就是欠定问题，坐姿、小幅腿部运动、上半身和脚弱相关动作尤其困难。
  • SMPL body only: Does not model SMPL-X hands/face or detailed hand articulation. It is avatar body tracking, not full whole-body mesh recovery.
    仅 SMPL 身体：不建模 SMPL-X 手/脸或精细手部关节；它是虚拟人身体追踪，不是完整 whole-body mesh recovery。

TL;DR

AvatarPoser (ETH Zurich + Meta Reality Labs, ECCV 2022) is an early sparse-HMD full-body avatar tracking method. It predicts 22-joint SMPL motion from only the headset and two hand/controller 6-DoF streams using a Transformer encoder, a global-motion Stabilizer, FK supervision, and post-network arm IK. On AMASS, the practical three-input setting reaches 4.18cm MPJPE vs VAE-HMD's 6.83cm, with reported network inference up to 662 FPS. Best viewed as a predecessor to HMD-Poser: strong sparse-sensor body tracking, but no RGB scene grounding and no SMPL-X hands/face.
