Dyn-HaMR: Recovering 4D Interacting Hand Motion from a Dynamic Camera

Zhengdi Yu, Stefanos Zafeiriou, Tolga Birdal
Imperial College London （WiLoR / HaMeR 同一实验室 Zafeiriou 组）
Hand Regression 4D Global Motion Bimanual SLAM Optimization Dynamic Camera arXiv 2025
📄 arXiv 🌐 Project Page
One-sentence problem: Existing hand reconstruction methods assume a static/weak-perspective camera and thus fail to recover global 4D trajectories when captured by a moving camera (e.g., egocentric) — can we disentangle hand motion from camera motion for two interacting hands simultaneously?
一句话问题：现有手部重建方法假设静态/弱透视相机，当相机移动（如自我中心场景）时无法恢复全局 4D 轨迹——能否同时为两只交互手部从相机运动中解耦手部运动？

Key Contributions

  • First 4D global hand motion from dynamic camera: Introduces a three-stage optimization pipeline that jointly estimates world-space MANO trajectories for both hands and the camera extrinsics, disentangling the two motion sources.
    首个动态相机下的 4D 全局手部运动：提出三阶段优化流水线，联合估计双手世界坐标系 MANO 轨迹和相机外参，解耦两种运动来源。
  • SLAM-based camera motion estimation: Uses DPVO SLAM to estimate relative camera transforms {R_t, τ^c_t}, then a learnable world scale factor ω disambiguates camera displacement magnitude vs. hand displacement in global space.
    基于 SLAM 的相机运动估计：使用 DPVO SLAM 估计相对相机变换 {R_t, τ^c_t}，然后通过可学习的世界尺度因子 ω 消除相机位移和手部位移大小的歧义。
  • Interacting-hand motion prior + biomechanical constraints: Stage III constrains the bi-manual interaction using HMP generative prior (NeMF-based) + joint angle bounds (L_ja) + bone length (L_bl) + palm geometry (L_palm) + penetration loss (L_pen), producing plausible interaction trajectories even under heavy occlusion.
    交互手部运动先验 + 生物力学约束：第三阶段使用 HMP 生成先验（基于 NeMF）+ 关节角度约束（L_ja）+ 骨长约束（L_bl）+ 掌部几何约束（L_palm）+ 穿透损失（L_pen），在严重遮挡下仍产生合理的交互轨迹。
  • Generative infilling for missed detections: In Stage I, missed/occluded frames are filled using HMP: optimize latent code z^h via slerp canonical interpolation, then decode full trajectories — avoids gaps and ensures temporal coherence before global optimization.
    缺失检测的生成式填充：第一阶段通过 HMP 填充缺失/遮挡帧：用球面线性插值优化隐变量 z^h，然后解码完整轨迹——在全局优化前避免间断并确保时序一致性。

Three-Stage Pipeline

Stage I — Initialization & Generative Infilling
ViTPose → 2D keypoints → HaMeR/ACR/IntagHand → per-frame MANO in camera space. Missed frames: HMP latent code z^h optimized via slerp in pose space → decoded trajectory fills gaps. Output: ^cQ^h (camera-space hand trajectory).
ViTPose → 2D 关键点 → HaMeR/ACR/IntagHand → 相机坐标系逐帧 MANO。缺失帧：通过姿态空间 slerp 优化 HMP 隐变量 z^h → 解码轨迹填补空缺。输出：^cQ^h（相机坐标系手部轨迹）。
Stage II — Global Optimization (Camera Motion Disentanglement)
DPVO SLAM → camera transforms C_t = {R_t, τ^c_t}. Compose: ^wq^h_t = C_t ⊙ ^cq^h_t. Optimize jointly: world trajectory ^wQ^h, world scale ω, camera extrinsics C_t, local hand pose/shape to match 2D reprojections. Loss: L_{2d} (Geman-McClure robust) + L_{smooth} + L_{cam} + L_J + L_β.
DPVO SLAM → 相机变换 C_t。组合：^wq^h_t = C_t ⊙ ^cq^h_t。联合优化：世界轨迹 ^wQ^h、世界尺度 ω、相机外参 C_t、手部局部姿态/形状以匹配 2D 重投影。损失：Geman-McClure 鲁棒 2D 重投影 + 平滑 + 相机 + 关节 + 形状。
Stage III — Interaction Prior Optimization
Re-optimize latent code z^h with full interaction constraints. Loss adds: L_prior (NLL under HMP) + L_pen (interpenetration) + L_bio (joint angles + bone lengths + palm). Only optimizes root orientation φ^h and translation τ^h in first 20 steps, then adds z^h for 200 steps.
以完整交互约束重新优化隐变量 z^h。新增损失：L_prior（HMP 下负对数似然）+ L_pen（穿透）+ L_bio（关节角度 + 骨长 + 掌部）。前 20 步仅优化根朝向 φ^h 和位移 τ^h，然后加入 z^h 优化 200 步。
MANO parameterization: q^h_t = {θ^h_t ∈ R^{3×15}, β^h ∈ R^10, φ^h_t ∈ R^3, τ^h_t ∈ R^3}. Shape β fixed across sequence. 778 vertices, 21 joints. World-to-camera: C_t = {R_t, τ^c_t}.
MANO 参数化：q^h_t = {θ^h_t ∈ R^{3×15}，β^h ∈ R^10，φ^h_t ∈ R^3，τ^h_t ∈ R^3}。形状 β 在整个序列中固定。778 个顶点，21 个关节。

Limitations

  • Limited time horizon: Optimization works for short sequences; extrapolating to long sequences requires regression-based approach (stated as future work).
    时序长度限制：优化适用于短序列；推广到长序列需要基于回归的方法（论文指出为未来工作）。
  • Slow inference: L-BFGS optimization is not real-time — not suitable for live AR/VR applications, unlike HaWoR (40ms/frame).
    推理速度慢：L-BFGS 优化非实时——不适合实时 AR/VR 应用，不如 HaWoR（40ms/帧）。
  • No body context: Hands are estimated independently of body pose — no kinematic anchoring at wrists. This is the key gap vs. HaWoR.
    无体态上下文：手部独立于体态估计——腕部无运动学锚点。这是相对 HaWoR 的主要差距。
  • No object interaction modeling: Future work explicitly mentions "incorporate object interactions" as next step.
    无物体交互建模：论文明确指出"整合物体交互"为下一步工作。

TL;DR

Dyn-HaMR (Imperial College, arXiv Dec 2024) is the first optimization-based method for 4D global bimanual hand motion from a dynamic (moving) camera. Three-stage pipeline: (1) hierarchical MANO init + generative infilling for occluded frames; (2) DPVO SLAM disentangles camera from hand motion + world scale ω; (3) HMP interaction prior + biomechanical + penetration losses refine interactions. On Ego-Exo4D: Jerk 5.26 vs HaMeR 195.45 (37× smoother), G-MPJPE 53.89mm vs 267.75mm (↓80%). From the same Zafeiriou lab as WiLoR; complementary to HaWoR (faster but no bimanual interaction prior).
