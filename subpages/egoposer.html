EgoPoser: Robust Real-Time Egocentric Pose Estimation from Sparse and Intermittent Observations Everywhere

Jiaxi Jiang, Paul Streli, Manuel Meier, Christian Holz
Department of Computer Science, ETH Zurich · Sensing, Interaction & Perception Lab

arXiv PDF Project GitHub
One-sentence problem: Can an MR headset recover full-body pose in large real scenes from only sparse head + hand/controller tracking, even when hands intermittently leave the headset field of view and users have different body sizes?
一句话问题：MR 头显能否只依赖头部 + 双手/控制器的稀疏追踪，在大场景中实时恢复全身姿态，并且在手部离开头显视场、用户体型不同的情况下仍然稳定？
Placement in this bank: EgoPoser is the direct sparse-HMD successor to AvatarPoser. It belongs under Ego-to-Exo SMPLX Regression -> Sparse Sensor / HMD, although the paper uses the first 22 SMPL-H/SMPL body joints and does not estimate articulated hands or face.
在本库中的定位：EgoPoser 是 AvatarPoser 之后的稀疏 HMD 全身姿态方法，归入 Ego-to-Exo SMPLX Regression -> Sparse Sensor / HMD。不过它实际输出的是 SMPL-H/SMPL 的 22 个身体关节，不估计精细手部或面部。

Key Contributions

  • Large-scene sparse-HMD pose estimation: Targets practical inside-out MR tracking where global headset/hand poses can be far from mocap-room origins, unlike training data concentrated near the origin.
    大场景稀疏 HMD 姿态估计：面向真实 inside-out MR 追踪，头显/手部的全局位置可能远离动捕房间原点，而不是只在训练数据附近活动。
  • Global-in-Local (GiL) motion decomposition: Keeps useful vertical/global trajectory cues while removing horizontal position dependence, making predictions robust to meter-scale and 50m-scale offsets.
    Global-in-Local (GiL) 运动分解：保留有用的竖直/轨迹信息，同时去除水平全局位置依赖，使模型对米级乃至 50m 偏移都稳定。
  • Realistic field-of-view modeling: Simulates whether each hand is actually visible from the headset based on relative hand/head geometry, instead of uniformly random hand masking.
    真实视场建模：根据手相对头部的几何关系判断手是否在头显视场内，而不是用均匀随机 masking 近似追踪丢失。
  • SlowFast temporal fusion: Uses both dense recent frames and sparse long-range frames over an 80-frame context, keeping Transformer cost similar to a 40-frame input.
    SlowFast 时序融合：在 80 帧上下文中同时使用高密度近邻帧和低密度长程帧，同时保持接近 40 帧输入的计算量。
  • Body-size adaptation: Adds calibration-based and calibration-free size-aware estimation so the avatar scale matches the user better and reduces floating/ground-penetration artifacts.
    体型适配：提供需要校准和免校准两种 size-aware 估计方式，让虚拟人比例更接近用户，并减少漂浮和穿地问题。

Method / Architecture

HMD + left/right hand 6-DoF poses -> GiL preprocessing -> FoV-aware masking -> SlowFast fusion -> Transformer encoder -> Multi-head decoder -> global body pose + SMPL-H shape

Input: 3-point tracking from the MR device: headset pose and two hand/controller poses. Each timestep uses positions, 6D rotations, linear velocity, and angular velocity; the final network input is 59 features per frame after decomposition.

Global-in-Local: Spatial normalization makes horizontal translation relative to the head but keeps vertical/global height information. Temporal normalization subtracts the first-frame horizontal translation within the window, preserving local trajectory shape without overfitting to absolute scene coordinates.

FoV modeling: A hand is masked only when its relative position falls outside the simulated headset horizontal/vertical field of view. When a hand is missing, EgoPoser uses predicted hand pose as a temporal reference for normalization.

Output: root/global localization, local body joint rotations, and shape parameters are fed into a differentiable SMPL-H body model. The method estimates body pose, not MANO hands or FLAME face.

输入：MR 设备提供的 3 点追踪：头显姿态和左右手/控制器姿态。每帧包含位置、6D 旋转、线速度、角速度；经过分解后每帧输入为 59 维。

Global-in-Local：空间归一化让水平位移相对头部，但保留竖直/身高相关的全局信息；时序归一化减去窗口首帧水平位移，保留局部轨迹形状，同时避免对绝对场景坐标过拟合。

FoV 建模：只有当手相对头部的位置落在模拟头显水平/竖直视场之外时才 mask。手缺失时，EgoPoser 使用预测手部姿态作为时序归一化参考。

输出：全局定位、局部身体关节旋转和体型参数进入可微 SMPL-H body model。该方法估计身体姿态，不输出 MANO 手部或 FLAME 面部。

Key lesson: For sparse-HMD body tracking, absolute world coordinates are a liability unless the representation is carefully decomposed. The most transferable insight for this paper bank is not the Transformer itself, but the input representation: keep motion cues that disambiguate pose while removing scene-origin dependence.
关键结论：对稀疏 HMD 全身追踪来说，如果不做表示分解，绝对世界坐标会成为严重负担。对本论文库最有迁移价值的不是 Transformer 本身，而是输入表示：保留能消歧身体姿态的运动线索，同时去掉场景原点依赖。

Limitations

  • No visual grounding: EgoPoser uses HMD/hand/controller tracking signals, not RGB video, so it cannot use scene contact, object interaction, visible body parts, or image evidence.
    没有视觉 grounding：EgoPoser 使用头显/手部/控制器追踪信号，不使用 RGB 视频，因此无法利用场景接触、物体交互、可见身体部位或图像证据。
  • Body only: Output is a SMPL-H/SMPL body representation with the first 22 body joints. It does not recover detailed MANO hands, face expression, or unified SMPL-X whole-body parameters.
    仅身体：输出是 SMPL-H/SMPL 的 22 个身体关节表示，不恢复 MANO 精细手部、面部表情或统一 SMPL-X whole-body 参数。
  • Still underdetermined: Inferring legs and pelvis from head + intermittent hands is intrinsically ambiguous; FoV modeling improves robustness but cannot remove all lower-body uncertainty.
    问题仍欠定：从头部 + 间歇手部推断腿部和骨盆本身存在歧义；FoV 建模增强鲁棒性，但不能消除所有下肢不确定性。
  • Mocap-driven training: The core model is trained on AMASS-style mocap, so real headset deployment still depends on tracking quality and the realism of simulated hand visibility failures.
    动捕驱动训练：核心模型仍基于 AMASS 式动捕训练，真实头显部署依赖追踪质量以及模拟手部可见性丢失是否足够真实。

TL;DR

EgoPoser (ETH SIPLab, ECCV 2024) is a real-time sparse-HMD full-body pose estimator for practical MR tracking. It predicts SMPL-H/SMPL body pose from headset + two intermittent hand/controller poses using Global-in-Local motion decomposition, realistic field-of-view masking, SlowFast temporal fusion, and body-size estimation. Its key result is large-scene robustness: on AMASS, AvatarPoser degrades from 4.18cm MPJPE at 0m offset to 31.67cm at 50m, while EgoPoser stays at 4.14cm. Best viewed as a robust sparse-sensor baseline between AvatarPoser and HMD-Poser, but not a full SMPL-X hand/face or RGB-grounded egocentric method.
