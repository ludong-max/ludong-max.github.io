One-sentence problem: How can we enrich the Nymeria egocentric dataset with standardized body formats (MHR / SMPL), dense 3D scene object annotations, instance-level object reconstructions, and new sensor modalities (wristband video, audio) to broaden its applicability for 3D scene understanding and body regression research?
一句话问题：如何通过添加标准化体态格式（MHR / SMPL）、密集 3D 场景物体标注、实例级物体重建以及新传感器模态（腕带视频、音频），来丰富 Nymeria 自我中心数据集，拓展其在 3D 场景理解和体态回归研究中的适用性？

What NymeriaPlus Adds

  • MHR body format NEW: Momentum Human Rig — 127-joint parametric model with shape β and pose θ ∈ ℝ²⁰⁴ (136 joint-angle + 68 bone-length params). Unlike SMPL-X, MHR explicitly decouples body surface from skeleton, allowing fine-grained bone-length control. Motion quality dramatically improved over original Nymeria annotations.
    MHR 体态格式 新增：Momentum Human Rig——127 关节参数化模型，形状参数 β 和姿态参数 θ ∈ ℝ²⁰⁴（136 关节角度 + 68 骨长参数）。与 SMPL-X 不同，MHR 显式解耦体表和骨骼，允许细粒度骨长控制。动作质量相比原始 Nymeria 大幅提升。
  • SMPL(-X) conversion NEW: Standard SMPL format derived from MHR, enabling direct compatibility with the broader body regression ecosystem (HMR, SMPLer-X, OSX, etc.).
    SMPL(-X) 转换 新增：从 MHR 派生的标准 SMPL 格式，可直接兼容更广泛的体态回归生态系统（HMR、SMPLer-X、OSX 等）。
  • 3D/2D bounding box annotations NEW: 10K+ fixed-taxonomy 3D OBBs (19 categories: chair, bed, couch, door, window, …) + ~12K open-set "Anything" objects across all 47 indoor venues. 2D projections computed automatically. Five densely-annotated venues provide 2,896 exhaustively labeled objects for benchmarking.
    3D/2D 边界框标注 新增：47 个室内场所中 1 万+ 固定分类 3D OBBs（19 类）和约 1.2 万个开放集"任意"物体。2D 投影自动计算。5 个密集标注场所提供 2,896 个详尽标注物体用于基准测试。
  • Instance-level 3D object reconstructions NEW: 4,547 high-quality 3D meshes from basemap recordings using the ShapeR algorithm (4 candidates per object, manual QA scored 1–5, threshold ≥ 3/5), out of 7,171 total candidates.
    实例级 3D 物体重建 新增：从底图录像中使用 ShapeR 算法生成 4,547 个高质量 3D 网格（每个物体 4 个候选，人工 QA 1–5 分，阈值 ≥ 3/5），共 7,171 个候选。
  • Basemap recordings NEW: 47 dedicated 5–15 min venue scans (one per indoor house) captured with Aria glasses for full 3D space coverage from diverse viewpoints.
    底图录像 新增：47 段专用场所扫描（每个室内房屋一段，每段 5–15 分钟），用 Aria 眼镜从多视角完整覆盖 3D 空间。
  • Wristband videos NEW: 600+ hours of wrist-perspective footage per participant (RGB + 2× grayscale forward-facing streams), fully anonymized.
    腕带视频 新增：每位参与者 600+ 小时腕部视角录像（RGB + 2× 灰度前向摄像头流），已完全匿名化。
  • Audio NEW: 600+ hours of spatial audio from Project Aria glasses microphones, covering both participants and observers, fully anonymized.
    音频 新增：Project Aria 眼镜麦克风采集的 600+ 小时空间音频，涵盖参与者和观察者，已完全匿名化。

Body formatMHR (127 joints) + SMPL conversion

Momentum Human Rig (MHR)

MHR is a parametric body model M(β, θ) → N-vertex mesh, designed at Meta for egocentric motion capture:

  • Skeleton: 127 joints with per-joint affine transforms computed via Linear Blend Skinning (LBS).
  • Shape β: Identity-space coefficients controlling global body shape.
  • Pose θ ∈ ℝ²⁰⁴: 136 joint-angle params (θᴾ) + 68 bone-length params (θᴵ).
  • Key difference from SMPL-X: Body surface is explicitly decoupled from skeleton — bone lengths are independently controllable, enabling better personalization to individual body proportions.

SMPL(-X) format is provided via conversion from MHR, enabling use with the standard body regression research ecosystem.

MHR 是 Meta 为自我中心动作捕捉设计的参数化人体模型 M(β, θ) → N 顶点网格：

  • 骨骼：127 个关节，通过线性混合蒙皮（LBS）计算每关节仿射变换。
  • 形状 β：身份空间系数，控制整体体型。
  • 姿态 θ ∈ ℝ²⁰⁴：136 个关节角度参数（θᴾ）+ 68 个骨长参数（θᴵ）。
  • 与 SMPL-X 的关键区别：体表与骨骼显式解耦——骨长可独立控制，更好地个性化适应个体体型比例。

同时提供从 MHR 转换的 SMPL(-X) 格式，可直接用于标准体态回归研究生态。

Bottom line: NymeriaPlus keeps the same recording scale as Nymeria but fixes the body annotation quality (MHR re-optimization), adds SMPL compatibility, and enriches the dataset with 3D scene context (objects, basemaps) and two new sensor streams (wristband video + audio). The SMPL conversion is the key enabler for using NymeriaPlus as a training/evaluation source in standard body regression research.

TL;DR

NymeriaPlus is a drop-in enrichment of the 2024 Nymeria dataset. Same 300h / 256 participants / 50 locations, but adds: (1) MHR + SMPL body formats with dramatically improved motion quality (wrist error ↓64.6%, foot sliding ↓71.9%); (2) 10K+ 3D object bounding boxes across 19 categories plus 4,547 instance meshes; (3) 600+ h wristband video; (4) 600+ h spatial audio; (5) 47 basemap venue scans. The SMPL conversion makes it directly usable in standard body regression pipelines for the first time.
