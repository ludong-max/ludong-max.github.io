WiLoR: End-to-end 3D Hand Localization and Reconstruction in-the-wild

Rolandos Alexandros Potamias, Jinglei Zhang, Jiankang Deng, Stefanos Zafeiriou
Imperial College London  ·  Shanghai Jiao Tong University
Hand Regression MANO ViT-Large Detection + Reconstruction In-the-Wild CVPR 2025
📄 arXiv ⭐ GitHub — rolpotamias/WiLoR 🌐 Project Page
One-sentence problem: Can a single end-to-end pipeline simultaneously localize multiple hands and reconstruct them in 3D at real-time speed, outperforming two-stage HaMeR-style systems on in-the-wild benchmarks?
一句话问题：单一端到端流水线能否同时实时定位多只手并进行 3D 重建，在野外基准上超越 HaMeR 式两阶段系统？

Key Contributions

  • End-to-end detection + reconstruction pipeline: A fully convolutional one-stage detector (DarkNet + PANet) runs at 138–175 FPS, feeding crops directly to a ViT-Large reconstruction head — no separate crop-resize step needed.
    端到端检测 + 重建流水线：全卷积单阶段检测器（DarkNet + PANet）以 138–175 FPS 运行，直接将裁剪区域送入 ViT-Large 重建头——无需独立的裁剪缩放步骤。
  • Multi-Scale Pose Refinement Module: Projects the coarse MANO mesh onto multi-resolution feature maps to extract per-vertex image-aligned features, predicting pose/shape residuals (Δθ, Δβ) — the key accuracy gain over single-pass regression.
    多尺度姿态精化模块：将粗略 MANO 网格投影到多分辨率特征图上，提取每顶点图像对齐特征，预测姿态/形状残差（Δθ, Δβ）——相比单次回归的关键精度提升。
  • WHIM — 2M in-the-wild training dataset: 2M+ hand images from 1,400 YouTube videos covering sign language, cooking, sports, and egocentric viewpoints, auto-annotated with an ensemble detector + MANO fitting pipeline with biomechanical constraints.
    WHIM——200 万野外训练数据集：来自 1,400 个 YouTube 视频的 200 万+ 手部图像，涵盖手语、烹饪、运动和自我中心视角，用集成检测器 + 生物力学约束 MANO 拟合自动标注。
  • SOTA on FreiHAND and HO3Dv2: PA-MPJPE 5.5mm on FreiHAND (vs HaMeR 6.0mm) and AUCJ 0.851 on HO3Dv2 (vs HaMeR 0.846), while being significantly faster at inference.
    FreiHAND 和 HO3Dv2 上的最优结果：FreiHAND 上 PA-MPJPE 5.5mm（vs HaMeR 6.0mm），HO3Dv2 上 AUCJ 0.851（vs HaMeR 0.846），同时推理速度明显更快。

Method Architecture

Stage 1 — Detection
Input image → DarkNet + PANet (multi-scale anchors-free) → bbox + hand-side + keypoints @ 138 FPS
Stage 2 — Reconstruction
Cropped hand → ViT-L (ViTPose pretrain · 1280-dim) + learnable tokens (pose / shape / camera) → coarse MANO (θ, β, t) → Multi-Scale Refinement (per-vertex image-aligned features → Δθ, Δβ) → final MANO mesh
Coarse-to-fine key insight: After the ViT regresses initial parameters, the predicted hand mesh is projected back onto multi-resolution feature maps from the deconvolutional layers. Per-vertex image-aligned features are sampled and used to predict residuals rather than absolute values — this forces the model to correct alignment errors geometrically rather than relying on global context alone. The multi-scale variant (vs single-scale) cuts PA-MPJPE from 6.0mm to 5.5mm on FreiHAND.
粗到精关键思路：ViT 回归初始参数后，预测的手部网格被投影回反卷积层产生的多分辨率特征图上。采样每顶点的图像对齐特征，用于预测残差而非绝对值——这迫使模型从几何上纠正对齐误差，而非单纯依赖全局上下文。多尺度版本（vs 单尺度）在 FreiHAND 上将 PA-MPJPE 从 6.0mm 降至 5.5mm。

HaMeR's advantage is the HInt egocentric evaluation benchmark (Ego4D/EPIC-KITCHENS/YouTube) which WiLoR does not directly report on. 结论：WiLoR 在所有受控基准上超越 HaMeR，同时内置检测功能。HaMeR 的优势在于 HInt 自我中心评估基准（Ego4D/EPIC-KITCHENS/YouTube），WiLoR 未直接在该基准上报告结果。

TL;DR

WiLoR (CVPR 2025, Imperial College) is a one-stage end-to-end pipeline for multi-hand detection + 3D MANO reconstruction. Its key innovation is a Multi-Scale Pose Refinement Module that samples per-vertex image-aligned features to predict residuals, paired with a built-in 138 FPS DarkNet detector. Trained on 2M+ WHIM YouTube images, it achieves SOTA on FreiHAND (PA-MPJPE 5.5mm, vs HaMeR 6.0mm) and HO3Dv2 (AUC 0.851, vs HaMeR 0.846). Direct successor / competitor to HaMeR with better accuracy and no external detector dependency.
